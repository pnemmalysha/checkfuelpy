Приложение для смартфона(android) написано на kivy.
Первичная БД - JSON файл.
В качестве БД SQLite.
Можно несколько аккаунтов.
Кнопки действий:
+ добавить запись
+ показать архив
+ добавить фото
+ редактировать запись
+ удалить запись
+ дополнительно
 + отобразить даты заправок на календаре
 + экспортировать архив чтобы можно было открыть на другом телефоне
 + ...
